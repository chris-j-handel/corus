# PART ONE: SIX CONDITIONS, AND A CLOSURE

## One relation, and one direction

Everything below runs on one relation, said in the plainest logic there is.

```
a < b        a carries to b
a ⊀ b        a does not carry to b
```

No equation appears anywhere here. No magnitude, no first term, no last term. One relation, read one way.

Three conditions on it, and a world with nothing standing still already carries them.

**Nothing stands.**

```
∀x ∃y ∃z :  y < x  ∧  x < z
```

Every term is carried to, and carries on. Read at its three faces: no first, no last, nothing standing apart.

**Carrying sets its own bound.**

```
∀x ∃y :  x < y  ∧  ∀z ( y < z  →  x ⊀ z )
```

A term x carries to, x does not carry past. Each relation is made at its own meeting, or it is not.

**A meeting makes a term neither brought.**

```
∀a ∀b ∀m ( a < m ∧ b < m )  →  ∃s :  m < s  ∧  a ⊀ s  ∧  b ⊀ s
```

Two carry into a meeting. A third leaves, carried to by neither of them.

## A ring, and no next

Three relations, and a third pointed back at a term already run:

```
a < b
b < c
c < a
```

Closed, they carry to their own start, and a start already run is no fresh term. A sequence returning to a term it has run carries no next. Standing stops as the next stops.

Turn one relation forward instead:

```
a < b
b < c
c < d
```

d is not a, b, or c, so nothing returns, and the sequence carries. Each fresh term calls a next relation, and each next relation calls a next term.

One turn of one relation parts the two. Pointing back closes. Pointing on carries.

A ring closes, and a quantifier running over an unbounded tail closes. A quantifier running over a whole unbounded tail forms one with no ring anywhere in it, and a paradox arrives with nothing referring to itself. A closure formed is the freeze, and a ring is one form of it.

**A hard problem is that closure, held still.** Two things a field keeps apart, unable to bring together, through every instrument brought to it. Particle or wave. Determined or free.
